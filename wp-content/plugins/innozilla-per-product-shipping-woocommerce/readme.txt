Per Product Shipping for WooCommerce - Define different per product shipping costs for products, based on customer location..

== Description ==

This is a WooCommerce extension that can set multiple, standalone or per product shipping method.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the '/wp-content/plugins/woocommerce-innozilla-shipping-per-product.zip' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin.
